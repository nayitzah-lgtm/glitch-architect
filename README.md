# Glitch Architect

## Setup Instructions for Rojo and Roblox Development

To set up a development environment for Rojo and Roblox, follow these steps:

### Prerequisites

1. **Roblox Studio**: Make sure you have Roblox Studio installed. You can download it from [here](https://www.roblox.com/create).

2. **Rojo**: Install Rojo by following these steps:
   - Open your terminal.
   - Run the following command to install Rojo:
     ```
     npm install -g rojo
     ```

### Creating a New Project

1. Create a new directory for your project:
   ```
   mkdir my-roblox-project
   cd my-roblox-project
   ```

2. Initialize a new Rojo project:
   ```
   rojo init
   ```

3. Add any necessary files and folders for your Roblox game.

### Running Rojo

To start syncing your project with Roblox Studio, run:
```
rojo serve
```

Then, open Roblox Studio and connect to your running Rojo server.
